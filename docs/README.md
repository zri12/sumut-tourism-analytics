# Tourism Clustering Web

Website Data Mining Pola Kunjungan Wisatawan Menggunakan Algoritma K-Means Clustering.

## Tech Stack
- Next.js
- Tailwind CSS
- JavaScript
- K-Means Clustering
- Vercel

## Fitur
- Dashboard
- Dataset
- Proses Clustering
- Hasil Clustering
- Rekomendasi

## Deployment
Deploy menggunakan Vercel dengan domain bawaan Vercel.
